Module Tetromino

module Tetromino: sig .. end

The representation of a tetromino.

type t 
type color = int * int * int 
val defaults : t list

defaults are the possible tetromino shapes.

val size : t -> int

size piece is the bounding box of piece.

val colors : color list

colors are the possible tetromino colors.

val max_size : int

max_size is the maximum size of the default tetrominoes.

val get_tet : string -> t

get_tet str returns the tetromino with name str. If it does not recognize the string it returns the i piece.

val value : t -> int -> int -> int -> (int * int * int) option

value piece rot c r is the grid value of piece where rot is the rotation from 0 to 3, and (c, r) are the coordinates.

val wall_kicks : t -> int -> [ `CCW | `CW ] -> (int * int) list

wall_kicks piece rot dir are the possible wall kick offsets of piece when in rot and rotating dir.
