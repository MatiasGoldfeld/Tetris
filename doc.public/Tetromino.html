Module Tetromino

module Tetromino: sig .. end
type tetromino =
| I
| J
| L
| O
| S
| T
| Z 
type t

The representation of a tetromino.

val defaults : t list

defaults is a t list of possible tetromino shapes.

val size : t -> int

size t is the bounding box of tetromino t.

val color : t -> (int * int * int) option

color tet is the color option of tet

val value : t -> int -> int -> int -> (int * int * int) option

value tet rot x y is a color option of the tetromino where rot is the rotation from 0 to 3, x is the x coordinate, and y is the y coordinate.
